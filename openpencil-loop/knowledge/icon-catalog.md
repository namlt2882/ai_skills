# Icon Usage

## Path Nodes
- Use "path" nodes, size 16-24px
- Use PascalCase + "Icon" suffix (e.g., "SearchIcon")
- System auto-resolves names to SVG paths

## Icon Font Nodes
- type: icon_font with iconFontName for lucide icons
- Sizes: 14/20/24px

## Common Lucide Icon Names
```
search, bell, user, heart, star, plus, x, check
chevron-right, chevron-left, chevron-down
settings, home, mail, phone, calendar, clock
eye, lock, shield, key
arrow-right, arrow-left, arrow-up, arrow-down
menu, more-horizontal, filter
image, camera, file, download, upload
edit, trash, share, copy
grid, list, layout
sun, moon, cloud, zap
users, message-circle, send
shopping-cart, credit-card
play, pause, volume
github, twitter, instagram, linkedin, youtube
alert-circle, alert-triangle, info, help-circle
```

**NEVER use emoji as icons.**
