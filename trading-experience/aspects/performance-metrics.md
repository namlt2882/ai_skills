# Performance Metrics Aspect

This aspect covers the measurement and evaluation of trading performance.

## Key Performance Indicators
- Return on investment (ROI)
- Sharpe ratio
- Maximum drawdown
- Win/loss ratio
- Average gain/loss ratio
- Profit factor
- Expectancy
- Calmar ratio

## Performance Analysis
- Time-based performance measurement
- Benchmark comparisons
- Attribution analysis
- Risk-adjusted returns
- Volatility measurements
- Correlation analysis with markets

## Connection Points
- Integrates with Risk Management for risk-adjusted performance evaluation
- Works with Trading Psychology to measure emotional impact on performance
- Connects to Portfolio Management for diversification effectiveness
- Feeds into Execution Methods for strategy optimization
