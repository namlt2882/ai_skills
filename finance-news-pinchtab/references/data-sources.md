## Data Source Reliability

### Credibility Tiers

| Tier | Sources | Characteristics |
|------|---------|-----------------|
| **Tier 1 (Highest)** | Official government sources, central banks, exchanges, securities regulators | Primary data, official releases |
| **Tier 2 (High)** | Reuters, Bloomberg, FT, WSJ, VNDIRECT DStock | Professional journalism, editorial standards |
| **Tier 3 (Medium)** | CNBC, regional news, trade publications, Vietnamese financial media | Good coverage, some bias possible |
| **Tier 4 (Lower)** | Blogs, opinion sites, social media | Requires verification |

### Verification Requirements

| Source Tier | Verification Needed |
|-------------|---------------------|
| Tier 1 | None - automatically credible (includes SSC, MOF, VSD, VSE, ISA, Competition Authority) |
| Tier 2 | Minimal - spot check |
| Tier 3 | Cross-reference with Tier 1-2 |
| Tier 4 | Full verification required |

## Data Access Methods

### PinchTab Scraping

**Use for:**
- Vietnamese news websites
- Global news websites (when RSS unavailable)
- Economic calendar (Investing.com)
- Social media profiles

**Configuration:**
- 4 parallel servers (ports 9867-9870)
- 1-2 tabs per source
- 30-second timeout
- 3 retries on failure

### RSS Feed Ingestion

**Use for:**
- Financial news (Reuters, Bloomberg, FT, CNBC, VietStock, CafeF)
- Official data releases (BLS, BEA, Eurostat, ONS, GSO, SBV, MPI, MOLISA, MOIT, MARD, MOC, MOT)
- Vietnamese regulatory announcements (SSC, MOF, VSD, VSE, ISA, Competition Authority)
- Economic calendar (Trading Economics)
- International organizations (IMF, World Bank, BIS)

**Configuration:**
- curl with 30-second timeout
- 3 retries on failure
- Parallel fetching
- XML/JSON parsing

### API Access

**Use for:**
- Real-time price data
- Market data feeds
- Economic data APIs

**Configuration:**
- API key management
- Rate limiting
- Error handling

### Source Failure Handling

| Failure Type | Action | Retry Strategy |
|--------------|--------|----------------|
| Timeout | Retry | 3 attempts, exponential backoff |
| HTTP 4xx | Skip | No retry (client error) |
| HTTP 5xx | Retry | 3 attempts, 5-second delay |
| Parse error | Log | Skip and continue |
| Empty response | Log | Continue with warning |

## Integration Notes

### Cross-Source Deduplication

- RSS items override website scraping for same source
- Social posts linked to articles by URL matching
- Calendar events deduplicated by indicator + date + country

### Language Considerations

- Vietnamese sources: Use Vietnamese for summaries
- Vietnamese official data (GSO, SBV, MPI): Use Vietnamese or translate as needed
- Vietnamese regulatory sources (SSC, MOF, VSD, VSE, ISA, Competition Authority): Use Vietnamese or translate as needed
- Global sources: Use English or translate to user language
- Official data: Typically in English, localize as needed

### Rate Limiting

| Source Type | Rate Limit | Notes |
|-------------|------------|-------|
| Website scraping | 1 request/second | Per domain |
| RSS feeds | 1 request/minute | Per feed |
| APIs | Per provider limits | Check documentation |
